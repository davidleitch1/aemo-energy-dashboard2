Renewable Energy Gauge - Iframe Embedding Example

Instructions:

  1. First, start the standalone gauge server:
    python standalone_renewable_gauge.py --port 5007 --allow-websocket-origin localhost:8000 yourdomain.com
  2. Then embed the gauge using an iframe as shown below
  3. Adjust the width and height as needed for your layout

Embedded Renewable Energy Gauge

Multiple Gauges Example

You can embed multiple instances if needed:

HTML Code:

<iframe 
    src="http://localhost:5007" 
    width="450" 
    height="400"
    title="Renewable Energy Gauge">
</iframe>

Security Considerations:

  • Use the --allow-websocket-origin parameter to specify allowed domains
  • For production, use HTTPS and proper domain configuration
  • Consider adding authentication if the data is sensitive
